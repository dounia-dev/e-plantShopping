# e-plantShopping 🌿

## 📌 Project Overview
This is a React-based application called **e-plantShopping**. 
It is an online plant shopping app where users can browse plants and add them to a cart.

## 🚀 Features
- View list of plants
- Navigate between pages
- Add items to cart
- View cart items

## 🛠️ Technologies Used
- React
- JavaScript
- HTML & CSS

## ▶️ How to Run
```bash
npm install
npm run preview
